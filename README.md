# Credit Risk Analysis Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard developed to analyze customer credit behavior, repayment trends, and default risk using a financial dataset. The dashboard combines business intelligence techniques, advanced DAX calculations, and Python-based customer segmentation to provide actionable insights for credit risk management.

---

## Business Problem

Financial institutions need to identify customers with higher default risk while maintaining profitable lending strategies. This dashboard helps stakeholders monitor customer payment behavior, evaluate credit utilization, and support data-driven credit decisions.

---

## Key Insights

- Identified customer default patterns and repayment behavior.
- Segmented customers into three behavioral clusters using PCA and K-Means.
- Evaluated payment ratios and credit utilization across customer groups.
- Provided recommendations for targeted credit policies and customer retention strategies.
